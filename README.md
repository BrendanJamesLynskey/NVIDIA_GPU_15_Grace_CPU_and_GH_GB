# Grace — NVIDIA's ARM Datacenter CPU and the GH200 / GB200 Superchips

Why NVIDIA built an ARM CPU and how it pairs with Hopper and Blackwell GPUs over the 900 GB/s NVLink-C2C coherent fabric. Covers the Grace Neoverse V2 cores, LPDDR5x memory, GH200 and GB200 superchips, NVL72, Extended GPU Memory (EGM), and the DGX Spark personal workstation. Includes an interactive superchip picker.

**Live site:** https://brendanjameslynskey.github.io/NVIDIA_GPU_15_Grace_CPU_and_GH_GB/
